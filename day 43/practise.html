1. Define database in your own words

A database is an organized place where information is stored so it can be easily saved, searched, updated, and managed. It helps computers handle large amounts of data quickly and accurately instead of using files or paper records.

2. Difference between data & database

Data

  • Data is raw information or facts.
  • It can be numbers, text, images, or symbols without meaning by itself
  • Examples: Raj,22,chennai

Database

  • A database is an organized collection of data
  • It stores data in a structured way so it can be easily accessed and managed.
  • Examples: A student database containing name,age,and city of many students

3. What is DBMS?

DBMS (Database Management System) is software that manages databases.

Examples:

  • MySQL
  • Oracle
  • SQL server

Responsibilities of DBMS

  • Data security
  • Data consistency
  • Multi-user access
  • Backup & recovery

4. What is SQL used for?

SQL (Structured Query Language) is used to create, read, update, and delete data.

Explanation:

SQL is the communication language between user and database.

Key points

  • works with tables
  • standard language
  • Easy to learn

5. Explain row & column

Row

  • A row is a horizontal line in a table
  • Each row represents one complete record
  • Examples:one students full details(ID,Name,Age).

Column

  • A column is a vertical line in a table
  • Each column represents one type of data or attribute
  • Examples: Name column, Age column, city column.
